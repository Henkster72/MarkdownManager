# Changelog

## 0.3.1

- Major translation effort: added DE/FR/PT UI strings, localized tutorials, and language-aware onboarding for WPM.
- HTML preview upgrades: caret footnotes, inline HTML support, strikethrough, themed buttons, and stronger link parsing.
- Editor UX improvements: AJAX save without refresh, save status indicator, replace modal, and clearer save error diagnostics.
- Theme polish: complementary secondary accent and preview layout refinements.

## 0.3

- Website publication mode for turning Markdown into a publish-ready content flow with metadata, publish states, and strict author/subtitle rules.
- Explorer sorting by date, title, or filename (with the date shown next to the slug).
- Settings upgrades for superusers: app title override and editor view toggles moved into Settings to free toolbar space.
- Mobile and layout refinements for the explorer overlay and header.
