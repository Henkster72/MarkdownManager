# Security Policy

## Supported Versions

This project does not currently publish formal version support windows.
If you are using a fork or a custom deployment, keep it updated and monitor
your own change history.

## Reporting a Vulnerability

Please report security issues privately.

- Email: henk@allroundwebsite.com
- Include: a clear description, affected files/paths, steps to reproduce,
  and any relevant logs or screenshots.
- Response time: we aim to acknowledge within 7 days.

Do not open public issues for security vulnerabilities.
